## テスト目的

本番環境での **24-48 時間連続稼働** を確認し、以下を検証：

1. **メモリリーク** - 連続実行でメモリ使用量が増加しないか
2. **CPU 安定性** - CPU 使用率が一定範囲内か
3. **プロセス安定性** - クラッシュやハング不具合がないか
4. **Pixhawk 通信継続** - ハートビート受信が途絶しないか
5. **ログファイルサイズ** - ログが肥大化していないか
